About This Website

This website is a simple project used to demonstrate my understanding of CI/CD automation, Git workflows, and deployment pipelines.

Every change committed to the repository automatically triggers a pipeline that builds and deploys the website.

This allows quick iteration and reliable deployments with minimal manual effort.

Website Version

Current Version: v1.0

This version contains a basic static website deployed using an automated pipeline. Future versions will introduce new images and updates pushed through Git commits.

CI/CD Pipeline Demonstration

Step 1: Developer pushes code to Git repository
Step 2: CI pipeline automatically triggers
Step 3: HTML files are validated and packaged
Step 4: Website is deployed to web server
Step 5: Users immediately see the updated version
