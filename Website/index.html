About This Website

This website demonstrates my understanding of CI/CD automation, Git workflows, and automated deployments.

Each code commit to the Git repository automatically triggers a pipeline which validates the build and deploys the website to the server.

This ensures reliable, repeatable deployments without manual intervention.
